## v1.1 — 2026-06-03

### 流程优化

- **新增：制作决策链**（分镜 Layer 2 通过后自动进入）
  - 第一问：是否需要生成剪辑方案（可选）
  - 第二问：配音方式选择（AI 手动 / AI API / 手动配音）；选 API 时系统提示提供接口文档和 API Key
  - 第三问：是否需要 AI 生成图片；选是后二选一（手动跑 / 调用 API）
  - 第四问：是否需要 AI 生成视频；选是后二选一（手动跑 / 调用 API）
  - 四问结束后输出收尾提示，明确宣告本期 AI 制作任务已全部完成，并引导发布后数据分析流程

- **优化：写脚本步骤同步建立数据分析文件夹**
  脚本确认时同步创建 `Projects/[频道名]/10_Analytics/[标题]/`，用户发布后知道直接将数据放入对应文件夹

- **移除：最终审核 / QC 放行检查触发词**
  原因：每步已有 Layer 1 + Layer 2 双层 Review，全部通过即等于全局质检通过，最终 QC 为重复劳动

- **新增：AI 视频生成路径**
  media 目录新增 `video/[期数]/`，支持 AI 视频生成素材归档

## v1.0 — 2026-04-01（首次交付）

### 系统基线

- 完整制片流程：10 个角色 + 7 个主步骤 + 数据分析闭环
- 5 种频道类型模板（知识讲解 / 历史故事 / 电影解说 / 真人教学 / Vlog）（注：v1.2 已更新为新五种类型）
- 双层 Review 机制（AI 自检 + 创作者判断）
- User_Overrides.md 个人覆盖层（支持按角色覆盖规则）
- 内容安全红线系统
- TTS 工具推荐（主推 MiniMax）

### 已知限制

- TTS 停顿标记的精细控制方案尚在验证中，下一版完善
- 图片生成工具需要用户自行配置

3. 查看本文件了解更新了什么
