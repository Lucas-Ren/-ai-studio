## 脚本结构模板

```
Hook（痛点/结果展示）
  → 展示做完之后是什么样子，或者点出最常见的错误
  → "大多数人在这一步搞错了……"

Promise（承诺）
  → 今天你会学到什么，多长时间能做到
  → 具体，可量化

Steps（步骤演示）
  → 每步：做什么 → 为什么这样做 → 常见错误
  → 演示镜头优先，旁白辅助

Shortcut（精华提炼）
  → 如果只记一件事，是什么？
  → 专业人士和新手的最大差距在哪里

CTA
  → 下一个相关技能/进阶内容预告
```

---

## 语气标准

- 像朋友教你，不是老师讲课
- "你"而不是"大家"
- 承认困难，但给出具体解决方法
- 演示镜头期间旁白要少——让画面说话

---

## Layer 2 质量判断标准

- Hook：你的目标观众看到第一句，会不会停下来？
- Steps：每个步骤，没看过视频的人能照着做吗？
- Shortcut：那一句"精华"，值得截图发朋友圈吗？
- 整体：看完，观众有没有立刻想试试的冲动？

---

## 角色权重

重度：编剧（步骤清晰度）+ 剪辑（演示节奏）
标准：制片人 + 宣发 + QC
轻度：美术指导 + 分镜师（真人拍摄，非AI生图）+ 配音（真人出声）
