## 脚本结构模板

```
Hook（前10秒）
  → 具体实物或历史场景入口，不是抽象概念
  → 核心悬念：观众原以为X，实际是Y
  → 权威背书：数字/机构/事件，建立可信度

Contrast（对比）
  → 两种命运/形态并列呈现，不解释，让画面说话

Naming（命名时刻）
  → 给这个现象一个让人记住的名字
  → 立即给出白话解释（"换句话说……"）

Mechanism × 2-3轮
  → 每轮：新案例 → 规律如何运作 → 与上轮连接

Mapping（个体迁移）
  → 直接用具体场景，不宣布"接下来映射到你"

Closing（收束）
  → 揭示系统规律 → 认知收束（冷感判断 + 反问）
  → 有操作空间 → 行动收束（2-3条原则 + 金句）
```

---

## 语气标准

- 冷静旁观，像在呈现物理事实
- 禁止共情句（"我们都经历过……"）
- 禁止公式句（"接下来映射到你身上……"）
- 举例优先：美国 / 欧洲 / 日本，避免中国案例

---

## Layer 2 质量判断标准

- 每隔30秒有一次"啊原来如此"的时刻？
- 能立刻指出脚本里最强的那一句？
- 语气像"冷静呈现事实的人"，而不是"想让你认同我的人"？
- 结尾：认知收束有脊背发凉感？行动收束原则足够具体？
