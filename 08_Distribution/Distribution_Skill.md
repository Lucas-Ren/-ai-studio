# 宣发 Skill 文件

**角色**：Distribution（宣发）
**对应流程**：STEP 6 — 宣发材料
**Review 节点**：REVIEW-8（宣发材料确认）

---

## 一、角色定义

宣发的工作是**让还不认识你的人，决定点进来**。封面、标题、简介是三秒内的销售行为。

---

## 二、输入

- 通过 Review 的脚本主文件（核心信息来源）
- 选题文档（对比结构、钩子元素）
- 触发词：`生成封面标题` 或在 STEP 6 自动触发

---

## 三、输出（交付物）

```
02_Scripts/YYYY-M-D_标题/
├── 封面描述_v1.txt
├── 封面描述_YouTube_v1.txt
├── 视频简介_v1.txt
└── 视频简介_YouTube_v1.txt
```

---

## 四、核心规则

**标题**：至少 2 个钩子元素（权威背书 / 具体数字 / 禁忌警告 / 悬念冲突）
**封面描述**：视觉反差清晰，人物/主体具体（含时代、身份）
**视频简介**：前两行包含最强钩子（折叠前只显示两行）

执行时读取 `Channel_Config.md` 中的平台设置和 CTA 规范。

### 视频简介文档格式（视频简介_v1.txt）

```
《标题》

候选标题 1：[标题]
候选标题 2：[标题]
候选标题 3：[标题]

---

[正文简介内容]
[CTA]
```

**标题写作规则**：
- 每个标题必须包含 ≥ 2 个钩子元素（权威背书 / 具体数字 / 禁忌警告 / 悬念冲突）
- 三个候选标题分别采用**不同的钩子策略**（不能三个都用同一种套路）
- 标题要制造"还没被回答的问题"——读完标题，观众脑子里要有一个疑问在悬着
- 禁止使用"揭秘" / "震惊" / "万万没想到"等流量词汇（显廉价）
- 具体胜过抽象：用人名、数字、地名，不用"某人" "很多钱" "很久以前"

---

## 五、双层 Review 清单

### 🤖 Layer 1 — AI 自检项

- [ ] 视频简介文档顶部有《标题》区块，含 3 个候选标题？
- [ ] 每个候选标题包含 ≥ 2 个钩子元素？
- [ ] 三个候选标题使用了不同的钩子策略（非同质化）？
- [ ] 每个标题读完后，观众脑子里有一个悬而未决的问题？
- [ ] 标题无"揭秘" / "震惊" / "万万没想到"等流量词？
- [ ] 封面主体有具体时代 + 身份（无"一个人"等模糊描述）？
- [ ] 视频简介正文前两行含核心悬念（非频道介绍）？
- [ ] 四个文件均已生成（国内封面 / YouTube 封面 / 国内简介 / YouTube 简介）？
- [ ] 简介末尾有频道 CTA？

### 👤 Layer 2 — 创作者判断项

- 三个候选标题里，哪一个让你"心里一动"？那就是它。
- 假装你不认识自己：刷到这个标题，你会停下来吗？
- 你的老粉看到这个封面，第一反应是"这就是他的风格"还是"他怎么了"？
- **Kennedy 品牌直觉**：如果答案是"可能会觉得不像"——修改，不发。
