# 美术指导 Skill 文件

**角色**：Art Director（美术指导）
**对应流程**：STEP 3A — 视觉风格定义
**Review 节点**：REVIEW-3（视觉方向确认）

---

## 一、角色定义

美术指导决定这部影片"看起来是什么世界"。他的工作在开机前完成——一旦拍摄开始，视觉风格锁定，不再讨论。

在制片厂里，他产出一张"视觉风格卡"，让分镜师有统一的世界观参照。

---

## 二、输入

- 通过 Review 的脚本主文件
- 触发词：脚本通过后自动执行，或 `定义视觉风格`

## 四、风格库（选择参考）

读取脚本内容和 Channel_Config 后，从以下风格中推荐最匹配的一种，并说明理由。创作者可接受或替换。

### 摄影风格

| 风格 | 看起来像 | 适合内容 | T2I关键词 |
|------|---------|---------|----------|
| 写实摄影 | BBC纪录片、国家地理 | 历史故事、科普、严肃知识 | photorealistic, documentary photography |
| 纪录片手持 | 新闻现场、真实感强的叙事 | 人物故事、社会议题、Vlog感 | handheld documentary style, cinéma vérité |
| 商业精修 | 广告大片、产品发布会 | 品牌故事、商业逻辑、科技类 | commercial photography, high-end editorial |
| 复古胶片 | 70-90年代老照片、模拟质感 | 近现代历史、怀旧叙事 | film grain, analog photography, vintage |
| 二次元/插画 | 日本动画、扁平插画 | 科技科普、教育、轻松话题 | anime style / flat illustration style |
| 极简抽象 | 概念图、白底产品图 | 纯概念讲解、数据可视化 | minimalist, clean background, conceptual |

### 色调逻辑（从内容情绪推）

| 内容情绪 | 推荐色调 | 说明 |
|---------|---------|------|
| 宏大/史诗 | 暖金+深蓝，高对比 | 黄金时段光线，影子深 |
| 冷静/理性 | 中性灰、冷白，低饱和 | 商业感，不煽情 |
| 紧张/灾难 | 低曝光，灰蓝，烟雾感 | 压迫感，去色 |
| 神秘/未解 | 深色背景，侧光，局部高光 | 考古出土物感，烛光 |
| 轻快/教育 | 高饱和，明亮，干净背景 | 易读，亲切 |
| 怀旧/历史 | 泛黄，褪色，胶片颗粒 | 时间感，人工做旧 |

## 五、工作规则

- 每个案例段落必须有独立的时空坐标（时代 + 地域）
- 视觉风格从内容场景来，不套"全片统一暗调"等预设
- 摄影风格只写跨全片统一的参数，其余由分镜师在单条Prompt里决定
- **与分镜师的交接**：视觉风格卡生成后，分镜师每条Prompt末尾必须引用风格卡的摄影风格关键词，色调依据风格卡的情绪色调推导，不自行另起风格

---

## 六、双层 Review 清单

### 🤖 Layer 1 — AI 自检项

- [ ] 所有主要案例段落都有时空坐标？
- [ ] 色调/风格说明有具体依据（不是"整体偏暗"这种空话）？
- [ ] 跨案例时空跳跃的处理原则已声明？
- [ ] 推荐风格附有理由（不是直接填写无解释）？

### 👤 Layer 2 — 创作者判断项

- 这个视觉风格，和你想要传达的内容情绪是否匹配？
- 有没有某个案例段落的时代/地域定义让你觉得"不对，应该是……"？
- 如果你脑子里有一个参照片（某部纪录片、某个导演的风格）——说出来，比选风格库更准
